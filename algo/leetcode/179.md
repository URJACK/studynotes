# [179. 最大数](https://leetcode-cn.com/problems/largest-number/)

本题最关键的地方在与，排序的时候，不要着眼于给<u>单个的元素赋予排序权值</u>！
<u>考虑两两组合赋予排序权值</u>也是可行的！

## 初始解法

```java
class Solution {
    public String largestNumber(int[] nums) {
        List<Unit> list = new ArrayList<>(nums.length);
        for (int i = 0; i < nums.length; i++) {
            list.add(new Unit(nums[i]));
        }
        Collections.sort(list);
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < list.size(); i++) {
            sb.append(list.get(i).toString());
        }
        return sb.toString();
    }

    class Unit implements Comparable<Unit> {
        /**
         * val = 210
         * data[0] = 0
         * data[1] = 1
         * data[2] = 2
         */
        int[] data;
        int length;


        public Unit(int val) {
            data = new int[10];
            if (val == 0) {
                length = 1;
            }
            while (val != 0) {
                int v = val % 10;
                this.data[length] = v;
                val = val / 10;
                length++;
            }
        }

        /**
         * a = 34  p = 2
         * b = 3   q = 1
         * <p>
         * a = 3
         * b = 3
         * <p>
         * a = 32
         * b = 3
         *
         * @param o
         * @return
         */
        @Override
        public int compareTo(Unit o) {
            int p = this.length;
            int q = o.length;
            while (p >= 0 && q >= 0 && this.data[p] == o.data[q]) {
                p--;
                q--;
            }
            if (p < 0) {
                p = 0;
            }
            if (q < 0) {
                q = 0;
            }
            return o.data[q] - this.data[p];
            //"34323 3432"
            //"3432 34323"
        }

        @Override
        public String toString() {
            StringBuilder sb = new StringBuilder();
            for (int i = length - 1; i >= 0; i--) {
                sb.append(data[i]);
            }
            return sb.toString();
        }
    }
}
```

## AC解法

```java
class Solution {
    public String largestNumber(int[] nums) {
        List<Unit> list = new ArrayList<>(nums.length);
        for (int i = 0; i < nums.length; i++) {
            list.add(new Unit(nums[i]));
        }
        Collections.sort(list);
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < list.size(); i++) {
            sb.append(list.get(i).toString());
        }
        /*清理*/
        dataClear(sb);
        return sb.toString();
    }

    /*
    处理 [0,0] 这种奇怪的输出为 '00' 的情况
    在这种条件下，也应该输出'0'
    */
    private void dataClear(StringBuilder sb) {
        for (int i = 0; i < sb.length(); i++) {
            if (sb.charAt(i) == '0') {
                sb.deleteCharAt(i);
                i--;
            } else {
                break;
            }
        }
        if (sb.length() == 0) {
            sb.append(0);
        }
    }

    class Unit implements Comparable<Unit> {
        String data;

        public Unit(int val) {
            data = String.valueOf(val);
        }

        /**
        修改了排序逻辑 权值的计算不再依靠各自的data 进行单独的比较
        而是通过组合二者的数据 得到两种组合 比较两种组合
        */
        @Override
        public int compareTo(Unit o) {
            String a = this.data + o.data;
            String b = o.data + this.data;
            return b.compareTo(a);
        }

        @Override
        public String toString() {
            return data;
        }
    }
}
```
